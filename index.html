Hotel Review Text Analytics

View My GitHub Profile

Data:

Dataset from http://www.kavita-ganesan.com/entity-ranking-data

We analyze only the hotel reviews from this dataset

The main objectives are:

* Parse and process the review data
* Perform text analytics on the data
* Topic modeling
* Key phrase extraction
* Sentiment analysis
* Store to MySQL and create dashboards

For performing the text analytics, we make use of two APIs - Azure text analytics and Text Processing.

Dashboards:

1. For each hotel in each city, plot the number of reviews over time
2. For each hotel in each city, plot the number of positive and negative reviews over time
3. For each city, create a word cloud with the key phrases
4. For each city, aggregate the sentiment
5. If possible, plot the hotels on Geo maps
6. For negative sentiment, pull the review title and key phrases
